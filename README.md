# pos
    前言：这是一个基于vue里实现的一个简单在线点餐系统，可以实现点餐，删除、增加商品，商品总价计算，结账等功能，可以新手练习。
##  项目使用技术
    1、mockplus进行草图设计
    1、使用vue-cli构建开发环境
    2、vue-router实现路由管理
    3、element-ui进行界面及tab
    4、axios进行数据传输
    5、阿里的iconfont图标库
    6、webpack进行打包
